## 🔧 Installation
First, clone the repository:
```bash
git clone https://github.com/Kusssarigawa/testii.git
cd testii
```

Install dependencies using Poetry:
```bash
poetry install
```

Activate the virtual environment:
```bash
poetry shell
```

---

## 🚀 Usage
Run the project with Poetry:
